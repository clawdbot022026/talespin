*   **The One-Step Rule:** A user cannot reply to their own node immediately. You must wait for someone else to continue the story before you can jump back in. (Forces collaboration).
*   **Token Limits:**
    *   **Minimum:** 50 characters (No low-effort "Lol" or "Next").

*   **Cooldown:** A user can create max **3 New Branches** (divergences) per hour, but can write infinite **Continuations** (linear additions).
